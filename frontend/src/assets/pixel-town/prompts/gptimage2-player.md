# gptimage2 Prompt — Player Character Sprite

## Asset: player-sprite-v1

**Target:** 224×24 PNG spritesheet (14 frames × 16×24 each)

**Prompt:**
```
original pixel art character sprite sheet, 16x24 pixels per frame,
14 frames in a single horizontal row on transparent background,
top-down 3/4 view RPG character,
cute anime-inspired girl with long flowing pastel pink-lavender hair,
soft fantasy flower crown head accessory,
gentle warm expression, simple dress in cream and soft pink,
1px dark blue-grey outline,
limited warm palette,
crisp pixels no anti-aliasing,

frame layout (left to right):
frames 1-2: idle facing down (subtle breathing animation)
frames 3-5: walk down (3-frame cycle)
frames 6-8: walk up (3-frame cycle)
frames 9-11: walk left (3-frame cycle)
frames 12-14: walk right (3-frame cycle)

style: cozy pixel RPG character, readable at tiny size,
warm gentle feeling, NOT realistic, NOT 3D,
no copyrighted game character, original design only
```

## Design Notes

The player character represents the user. Key visual traits:
- Long hair is the most recognizable feature at small size
- Flower crown creates a distinct silhouette
- Pastel pink/lavender palette makes her stand out from NPC agents
- Gentle, approachable feeling — this is a cozy town, not a battle game
